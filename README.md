## App Setup

In the new Terminal window, run the following commands:

`git clone git@github.com:sydnerd/nytimes-takehome.git`  
`cd nytimes-takehome`  
`npm install`  
`npm start`  

Open a window in your web browser of choice, and visit `localhost:3000` to view the app!
